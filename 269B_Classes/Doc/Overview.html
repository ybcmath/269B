This site provides links to component source files and other supporting material for Math 269B .

Notes:

For convenience, it is possible to download all of the sources in a single zip file. The directory structure of the uncompressed zip file is the recommended structure for the component sources. If one of the component sources is updated, links are povided that enable one to download only the component that has changed.

The recommended placement of the aggregate component directory (here named (269B_Classes) and the 269B_Samples directory is 

      Your_269B_Directory
================================================== 
    |           |                |              |
Your_Assign1 Your_Assign_2   269B_Samples    269B_Classes
                             =========       =========
                                ****           ****
